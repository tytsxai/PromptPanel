## 已清理

| 路径 | 性质 | 恢复方式 |
| --- | --- | --- |
| `.build/` | SwiftPM 构建缓存、依赖 checkout、模块缓存 | `swift build`、`swift test` 或打包脚本会重新生成。 |
| `dist/` | 本地打包和发布预检输出 | `./scripts/build-app.sh` 或 `./scripts/release-readiness.sh` 会重新生成。 |
| `.swiftpm/` | SwiftPM 本地状态目录 | SwiftPM 会按需重新生成。 |
| `Package.resolved` | 当前被 `.gitignore` 忽略的本地依赖解析结果 | SwiftPM 会按需重新解析生成。 |

- 项目入口：`README.md`、`README.zh-CN.md`、`CHANGELOG.md`、`LICENSE`。

- 源码与验证入口：`Sources/`、`Tests/`、`scripts/`。

- 公开检索元数据短入口：`llms.txt`、`codemeta.json`。
- GitHub 协作入口：`.github/`。
