# PromptPanel 兼容性回归记录

更新时间：`2026-05-07`

用途：

- 固定记录真实目标应用中的唤起、焦点恢复、自动粘贴和复制兜底表现
- 配合应用内“最近执行记录”的失败原因字段，定位兼容性边界

记录说明：

- 每次回归尽量使用同一批词条和相近机器状态
- 自动粘贴是否成功，以目标应用里内容是否真实落入输入框为准
- 如果出现 `target_app_not_restored`，优先记录是否与切桌面、切 Space、窗口层级或应用激活策略有关
- 热键相关记录优先级最高，先记录“按下后有没有秒开、搜索框有没有立刻可输入”
