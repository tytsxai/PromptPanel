# PromptPanel 项目介绍

更新时间：`2026-05-07`

本文用于给新用户、贡献者、搜索引擎和 AI 检索系统提供一份稳定的项目入口说明。更完整的公开入口在根目录 [README](../README.md) 和 [README.zh-CN](../README.zh-CN.md)。

## 一句话定位

PromptPanel 是一款开源、本地优先的 macOS Prompt 管理器和代码片段快捷面板。用户按下全局快捷键后，可以在原生 `NSPanel` 里搜索本地 Prompt 库，并把选中的多行文本粘贴到 ChatGPT、Claude、Cursor、Copilot、VS Code、终端、浏览器或任意当前输入框。

## 解决的问题

PromptPanel 解决的是一个高频、具体、长期存在的问题：

```text
同一批 Prompt / 指令 / 模板每天被反复复制、搜索、粘贴，但普通笔记工具太慢，浏览器插件又只能在单一网站里工作。
```

它把主链路收敛成：

```text
全局快捷键 -> 搜索 -> 回车 -> 先写剪贴板 -> 尝试自动粘贴 -> 失败时保留剪贴板兜底
```

这个链路的关键不是主窗口 CRUD，而是“任何 app 中快速召回并可靠落地文本”。

## 目标用户

- 重度 ChatGPT、Claude、Gemini、Perplexity 用户。
- Cursor、Copilot、Aider 等 AI coding 工具用户。
- 经常复用代码评审模板、排障清单、提交说明模板、命令片段的开发者。
- 同时维护多个客户、仓库或上下文的独立开发者、顾问和技术负责人。
- 需要维护标准回复、规范模板、状态更新、PRD 片段的 PM 和技术写作者。

## 核心能力

| 能力 | 当前实现 |
| --- | --- |
| 全局快捷键 | `KeyboardShortcuts` 捕获系统热键，默认 `⌥2`，可在设置中修改。 |
| 快捷面板 | AppKit `NSPanel` + SwiftUI 内容，面板可固定、可记住位置和尺寸。 |
| 搜索 | SQLite FTS5 按标题和正文搜索，支持当前项目 + `通用项目` 混合结果。 |
| 标签过滤 | 快捷面板支持 `#tag` 语法，在 FTS 结果上做本地二次过滤。 |
| 项目隔离 | 每条词条属于一个项目；`通用项目` 永远可见且不可删除。 |
| 执行 | 先写系统剪贴板，再尝试恢复目标 app 并派发 `Command-V`。 |
| 兜底 | 无辅助功能权限或目标 app 未恢复焦点时，保留剪贴板并提示手动粘贴。 |
| 诊断 | 执行日志记录权限、目标 app、失败原因和耗时，不记录词条正文。 |
| 本地数据 | SQLite 数据库位于 `~/Library/Application Support/PromptPanel/promptpanel.db`。 |
| 发布 | SwiftPM 构建，脚本打 `.app` 和 zip，支持 Developer ID 签名、公证和 Sparkle。 |

## 项目边界

PromptPanel 是单用户、本地、轻量的桌面工具，不是 SaaS、不是团队知识库、不是浏览器插件、也不是工作流编排平台。

永久不做：

- 云同步。
- 团队协作。
- 多用户权限系统。
- 复杂自动化工作流编排。
- 浏览器插件替代主应用。

这些不是暂缓项，而是产品边界。边界来自 [PRD](./项目快贴-PRD.md)，贡献者应优先保留主链路速度和本地可信度。

## 维护原则

- 文档要描述当前代码和脚本的真实行为，不把计划写成现状。
- 修改功能时同步更新 README、docs、FAQ、llms 索引和元数据。
- UI 改动先对齐 `frontend-draft/`，再落 SwiftUI/AppKit。
- 发布能力优先放在脚本和 CI，不塞进主应用运行时。
- 任何影响数据、粘贴、权限、发布、恢复的改动都要配测试或可复现验证。
