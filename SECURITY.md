# Security Policy

## Supported versions

PromptPanel is pre-release software. Security fixes are handled on `main` until the first tagged public release.

## Reporting a vulnerability

Do not open a public issue for a security problem.

Email the maintainer at `wwtvn1937@gmail.com` with:

- A short description of the issue.
- Reproduction steps.
- The affected version or commit.
- Any relevant logs with private prompt content removed.

You should receive an acknowledgement within 7 days. If the issue is confirmed, we will coordinate a fix before public disclosure.

## Data handling expectations

PromptPanel is local-first. Prompt content should stay on the user's Mac except when the user explicitly copies or pastes it into another app. Security reports involving prompt leakage, unexpected network access, unsafe restore behavior, or permission escalation are treated as high priority.
